    Arduino, Micropython, Automatyka domowa, Inteligentny dom

    Jeśli termin IoT (nie) jest Tobie obcy i zabrakło Ci czasu lub pomysłu, żeby zagłębić się w temat to masz teraz okazję. Arkadiusz Baraniecki, Artur Maćkowiak i Michał Śmigielski pokażą Wam jak rozpocząć przygodę z układem esp8266.

    Poruszane zagadnienia

    1. Arduino
    2. Micropython
    3. Automatyka domowa
    4. Inteligentny dom

    Wymagania w stosunku do uczestników

    Must have:
    Podstawowa znajomość C lub Pythona
    Laptop z portem USB (w przypadku nowych MacBook im prostsza przejściówka tym lepiej)
    Znajomość terminala (dotyczy również windows)
    Zainstalowany program do łączenia się portem seryjnym (np. screen dla linux i osx, putty dla Windows)
    Git dostępny z poziomu konsoli

    Nice to have:
    Python 3.6 (w przypadku windows konieczne zaznaczenie opcji “For all users”)
